# 🚀 Rapidoss

Application de logistique et livraison à la demande à Kinshasa, RDC.

## 🎨 Design

- **Palette** : Noir profond (#121212) avec accents verts (#29BA1F, #098C04)
- **Style** : Glassmorphisme futuriste avec animations fluides
- **Icônes** : Système d'icônes personnalisées SVG
- **Responsive** : Optimisé mobile-first

## 🛠️ Technologies

- Next.js 15 (App Router)
- React 19.2
- Tailwind CSS 4
- Framer Motion
- Leaflet + React Leaflet
- TypeScript

## 📦 Installation

```bash
npm install
npm run dev
```

## 🚀 Déploiement Vercel

```bash
# CLI
npm i -g vercel
vercel --prod

# Ou connectez GitHub à Vercel pour déploiement automatique
```
